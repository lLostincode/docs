---
title: 'List Knowledge Bases'
openapi: 'GET /api/v1/knowledge-bases/'
---

Retrieve all knowledge bases for the authenticated user's company.

## Query Parameters

<ParamField query="status" type="string">
  Filter by status: `pending`, `running`, `completed`, `failed`
</ParamField>

<ParamField query="processing_engine" type="string">
  Filter by processing engine: `milvus`, `cognee`
</ParamField>

<CodeGroup>
```bash cURL
curl -X GET "https://api-be.atthene.com/api/v1/knowledge-bases/?status=completed" \
  -H "x-api-key: YOUR_API_KEY"
```

```python Python
import requests

response = requests.get(
    "https://api-be.atthene.com/api/v1/knowledge-bases/",
    headers={"x-api-key": "YOUR_API_KEY"},
    params={"status": "completed"}
)
knowledge_bases = response.json()
```

```javascript JavaScript
const response = await fetch('https://api-be.atthene.com/api/v1/knowledge-bases/?status=completed', {
  headers: {
    'x-api-key': 'YOUR_API_KEY'
  }
});
const knowledgeBases = await response.json();
```
</CodeGroup>
