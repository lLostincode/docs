---
title: 'Check Knowledge Base Status'
openapi: 'GET /api/v1/knowledge-bases/{kb_id}/status/'
---

Monitor processing status and health of a knowledge base.

<ParamField path="kb_id" type="string" required>
  The knowledge base ID
</ParamField>

<CodeGroup>
```bash cURL
curl -X GET https://api-be.atthene.com/api/v1/knowledge-bases/kb_123/status/ \
  -H "x-api-key: YOUR_API_KEY"
```

```python Python
import requests

response = requests.get(
    "https://api-be.atthene.com/api/v1/knowledge-bases/kb_123/status/",
    headers={"x-api-key": "YOUR_API_KEY"}
)
status = response.json()
```

```javascript JavaScript
const response = await fetch('https://api-be.atthene.com/api/v1/knowledge-bases/kb_123/status/', {
  headers: {
    'x-api-key': 'YOUR_API_KEY'
  }
});
const status = await response.json();
```
</CodeGroup>

## Status Values

<AccordionGroup>
  <Accordion title="pending">
    Knowledge base created but processing not started
  </Accordion>
  <Accordion title="running">
    Collections are being processed and indexed
  </Accordion>
  <Accordion title="completed">
    All collections processed successfully
  </Accordion>
  <Accordion title="failed">
    Processing failed - check error details
  </Accordion>
</AccordionGroup>
