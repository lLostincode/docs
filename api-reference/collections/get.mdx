---
title: 'Get Collection'
openapi: 'GET /api/v1/collections/{collection_id}/'
---

Retrieve collection details including all data sources.

<ParamField path="collection_id" type="string" required>
  The collection ID
</ParamField>

<CodeGroup>
```bash cURL
curl -X GET https://api-be.atthene.com/api/v1/collections/coll_123/ \
  -H "x-api-key: YOUR_API_KEY"
```

```python Python
import requests

response = requests.get(
    "https://api-be.atthene.com/api/v1/collections/coll_123/",
    headers={"x-api-key": "YOUR_API_KEY"}
)
collection = response.json()
```
</CodeGroup>
