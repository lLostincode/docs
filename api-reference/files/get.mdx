---
title: 'Get File'
openapi: 'GET /api/v1/files/{file_id}/'
---

Retrieve file details and metadata.

<ParamField path="file_id" type="string" required>
  The file ID
</ParamField>

<CodeGroup>
```bash cURL
curl -X GET https://api-be.atthene.com/api/v1/files/file_123/ \
  -H "x-api-key: YOUR_API_KEY"
```

```python Python
import requests

response = requests.get(
    "https://api-be.atthene.com/api/v1/files/file_123/",
    headers={"x-api-key": "YOUR_API_KEY"}
)
file_info = response.json()
```

```javascript JavaScript
const response = await fetch('https://api-be.atthene.com/api/v1/files/file_123/', {
  headers: {
    'x-api-key': 'YOUR_API_KEY'
  }
});
const fileInfo = await response.json();
```
