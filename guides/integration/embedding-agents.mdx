# Chatbot Embedding Configuration

Configure your public chatbot by passing props to the chat component. All parameters are optional except `connectionKey`.

## Required Parameters

 \<ParamField path="connectionKey" type="string" required\> Your unique chatbot connection identifier. This key links the embedded chat to your chatbot configuration. 

```
connectionKey: "e97e32ea-579f-448c-a953-21586dccbc28"
```

 \</ParamField\> 

## Display & Branding

 \<ParamField path="type" type="string" default="inPlace"\> Determines how the chatbot appears on your page. 

- `inPlace` - Embedded directly in the page
- `popup` - Floating chat button with expandable window \</ParamField\>

 \<ParamField path="theme" type="object" default="{}"\> Custom theme object for styling the chatbot interface. Override default colors, fonts, and other visual elements. 

```
theme: {
  primaryColor: "#007bff",
  backgroundColor: "#ffffff"
}
```

 \</ParamField\> \<ParamField path="font" type="string" default=""\> Custom font family for the chatbot interface. Specify any web-safe font or Google Font. 

```
font: "Inter, sans-serif"
```

 \</ParamField\> \<ParamField path="images" type="object" default="{}"\> Custom images for the chatbot UI (avatar, logo, etc.). 

```
images: {
  avatar: "https://example.com/avatar.png",
  logo: "https://example.com/logo.svg"
}
```

 \</ParamField\> 

## Messages & Content

 \<ParamField path="welcomeMessage" type="string" default=""\> Initial greeting message displayed when the chat loads. 

```
welcomeMessage: "Hello! How can I help you today?"
```

 \</ParamField\> \<ParamField path="placeholder" type="string" default=""\> Placeholder text in the message input field. 

```
placeholder: "Type your message here..."
```

 \</ParamField\> \<ParamField path="hintQuestions" type="string[]" default="[]"\> Array of suggested questions displayed to guide users. 

```
hintQuestions: [
  "What are your business hours?",
  "How do I reset my password?",
  "Tell me about pricing"
]
```

 \</ParamField\> \<ParamField path="disclaimer" type="string" default=""\> One-time disclaimer message shown before the chat starts. 

```
disclaimer: "This chatbot provides general information only."
```

 \</ParamField\> \<ParamField path="persistentDisclaimer" type="string"\> Disclaimer that remains visible throughout the conversation (typically shown at the bottom of the chat). 

```
persistentDisclaimer: "AI responses may not always be accurate."
```

 \</ParamField\> 

## Audio Configuration

 \<ParamField path="enableAudio" type="boolean" default="true"\> Enable or disable voice output capabilities. \</ParamField\> \<ParamField path="playAudioByDefault" type="boolean" default="true"\> Whether audio playback is enabled by default when `enableAudio` is true. Users can toggle this setting. \</ParamField\> \<ParamField path="voiceConfig" type="object"\> Configuration for ElevenLabs text-to-speech. Only applies when `enableAudio` is enabled. \<Expandable title="Voice Configuration Properties"\> \<ParamField path="model_id" type="string" default="eleven_turbo_v2_5"\> ElevenLabs TTS model. Available options: - `eleven_turbo_v2_5` - Low latency, recommended for streaming (default) - `eleven_flash_v2_5` - Lowest latency option - `eleven_flash_v2` - Fast alternative - `eleven_turbo_v2` - Previous turbo version - `eleven_multilingual_v2` - Multi-language support - `eleven_multilingual_ttv_v2` - Multi-language with voice cloning - `eleven_ttv_v3` - Advanced voice cloning - `eleven_v3` - Latest high-quality model \</ParamField\> 

```
<ParamField path="voice_id" type="string" default="JBFqnCBsd6RMkjVDRZzb">
  Voice character ID. Available voices:
  - `JBFqnCBsd6RMkjVDRZzb` - George (default)
  - `L0Dsvb3SLTyegXwtm47J` - Archer: Grounded British male
  - `g6xIsTj2HwM6VR4iXFCw` - Jessica: Empathetic and expressive
  - `OYTbf65OHHFELVut7v2H` - Hope: Bright and uplifting
  - `HDA9tsk27wYi3uq0fPcK` - Stuart: Professional Australian
  - `1SM7GgM6IMuvQlz2BwM3` - Mark: Relaxed and laid back
  - `PT4nqlKZfc06VW1BuClj` - Angela: Raw and relatable
  - `vBKc2FfBKJfcZNyEt1n6` - Finn: Tenor pitched
  - `56AoDkrOh6qfVPDXZ7Pt` - Cassidy: Engaging and energetic
  - `NOpBlnGInO9m6vDvFkFC` - Grandpa Spuds: Distinctive character voice
</ParamField>

<ParamField path="speed" type="number" default="1.2">
  Speaking speed. Range: `0.7` (slower) to `1.2` (faster). Extreme values may affect quality.
</ParamField>

<ParamField path="stability" type="number" default="0.1">
  Voice consistency. Range: `0.0` to `1.0`
  - `0.0-0.4` - More emotional and dramatic
  - `0.6-1.0` - More monotone and consistent
</ParamField>

<ParamField path="similarity_boost" type="number" default="0.8">
  How closely the AI adheres to the original voice. Range: `0.0` to `1.0`. Higher values produce results closer to the original voice.
</ParamField>

<ParamField path="use_speaker_boost" type="boolean" default="false">
  Boost similarity to original speaker. Increases latency. Not available for `eleven_v3` model.
</ParamField>

<ParamField path="style_exaggeration" type="number" default="0.0">
  Amplify the style of the original speaker. Range: `0.0` to `1.0`. Increases latency and may reduce stability. Recommended: `0.0`
</ParamField>
```

 \</Expandable\> 

```
voiceConfig: {
  model_id: "eleven_turbo_v2_5",
  voice_id: "JBFqnCBsd6RMkjVDRZzb",
  speed: 1.2,
  stability: 0.1,
  similarity_boost: 0.8,
  use_speaker_boost: false,
  style_exaggeration: 0.0
}
```

 \</ParamField\> 

## User Management

 \<ParamField path="userToken" type="string" default=""\> Optional user identifier for tracking conversations and maintaining context across sessions. 

```
userToken: "user_abc123"
```

 \</ParamField\> 

## Example Configuration

```
<ChatWidget
  connectionKey="e97e32ea-579f-448c-a953-21586dccbc28"
  type="popup"
  welcomeMessage="Welcome! How can I assist you?"
  placeholder="Ask me anything..."
  hintQuestions={[
    "What services do you offer?",
    "How do I get started?"
  ]}
  enableAudio={true}
  playAudioByDefault={true}
  voiceConfig={{
    model_id: "eleven_turbo_v2_5",
    voice_id: "OYTbf65OHHFELVut7v2H", // Hope voice
    speed: 1.1,
    stability: 0.3
  }}
  theme={{
    primaryColor: "#4F46E5"
  }}
/>
```
